## 🚀 Cómo ejecutar el proyecto localmente

1. **Clona el repositorio:**

   ```bash
   git clone https://github.com/CardonaAndres/my-portfolio.git
   ```

2. **Instala las dependencias:**

   ```bash
   npm install
   ```

3. **Ejecuta el entorno de desarrollo:**

   ```bash
   npm run dev
   ```
